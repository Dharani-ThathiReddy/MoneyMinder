# MoneyMinder

## Overview
MoneyMinder is a full-stack web application designed to help users manage personal finances efficiently. It goes beyond basic expense tracking by providing goal planning, investment management, and financial insights.

---

## Features

### Dashboard
- Displays total balance, expenses, goals, and investments
- Visual representation using charts

### Expense and Income Tracking
- Add, edit, and delete transactions
- Categorize expenses
- Upload CSV/Excel files for automatic categorization and insights

### Goal Management
- Set short-term and long-term financial goals
- Track progress visually

### Saving Suggestions
- Provides suggestions based on user spending patterns

### Investment Management
- Add and track investments (stocks, cryptocurrency, etc.)
- Real-time price updates using Finnhub API
- Displays portfolio value

### Financial Reports
- Download reports in PDF format
- Includes expenses, goals, and investment summaries
